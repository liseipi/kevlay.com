科研和技术

Kevlay不仅拥有自己的研发团队，我们自主创新的产品与国际上其它优秀供应商亦保持着紧密长远的合作，以满足用户广泛而又独特的要求。Kevlay已完成了智能电网关于振动、噪音及高压设备绝缘测试和诊断的实时需求，产品已成功应用于上百个6KV—500KV变电站关键设备监测的工程实施，应用覆盖各级电网、电厂、石油化工、冶金、水利等领域。Kevlay将不断地收集、整理来自用户的需求和经验，为未来的发展构筑一个坚实的基础平台。
